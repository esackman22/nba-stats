Workflow:
    1. Pull in games from the given date
        a. Ask for date in CLI. If not in the correct format, prompt for another input.
    2. Pull plays for each game
    3. Transform plays dataframe to be ready for output
        a. Need max play_id from DB
        b. Need to make sure we meet all foreign key constraints
    4. Output plays dataframe to a csv
    5. Move csv to S3
    6. Import data from S3 using lambda function to Postgres DB
